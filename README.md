# MarketCap Metrics

## APIs

The following APIs are used in this project:

- **[Alpha Vantage](https://www.alphavantage.co/):** Used for retrieving stock data.
- **[CoinMarketCap](https://coinmarketcap.com/api/):** Used for retrieving cryptocurrency data.
